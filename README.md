# Rustic Translator

Translation management system for ERPNext that allows Translation Managers to edit language translation CSV files through a modern Vue.js web interface.

## Features

- Edit ERPNext/Frappe app translations via web interface
- Automatic backup management
- Audit trail for translation changes
- Safe cache clearing and migration

## Installation

```bash
bench get-app rustic_translator
bench --site your-site install-app rustic_translator
```
